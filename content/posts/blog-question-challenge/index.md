## Sejak kapan anda mulai ngeblog?
Saya kurang begitu ingat tapi saya mulai membeli domain pertama saya pada tahun 2018 akhir dengan domain jundimubarok.com tapi karena semakin mahal, akhirnya saya pindah ke jundi.web.id yang 4x lebih murah dibanding ekstensi .com. Tulisan pertama saya tidak ingat sama sekali karena tidak ada arsip, dan saya berpindah platform.

## Platform apa yang Anda gunakan untuk mengelola blog Anda dan mengapa Anda memilihnya?
Awal punya domain saya menggunakan platform wordpress yang memiliki tema yang melimpah, plugin, serta kustomisasi yang fleksibel, bahkan saya sampai membeli 2 tema premium yang bisa diinstall di unlimited website. Sampai pada akhirnya saya tertarik untuk menggunakan hugo, platform generator website statis yang menggunakan bahasa Go, walaupun pada akhirnya saya melihat dokumentasi karena pemahaman saya yang minim akhirnya sampai saat ini saya masih menggunakannya dengan alasan tidak ada database, sehingga tidak membebani server seperti halnya wordpress. sebenarnya alasan yang utama adalah minimnya modal sehingga website statis menjadi pilihan.

## Apakah Anda pernah ngeblog di platform lain sebelumnya? 
belum pernah, tapi akan saya coba

## Bagaimana cara Anda menulis postingan? Misalnya, di alat pengeditan lokal, atau di panel/dasbor yang merupakan bagian dari blog Anda? 
saya menuliskannya di dalam editor markdown yang saya coba beberapa aplikasi yang cocok, pada akhirnya langsung dituliskan di dalam visual studio code, dan saat ini menggunakan google docs sehingga bisa dilanjutkan di smartphone jika tidak sedang menggunakan komputer, setelah itu barulah tulisan saya pindahkan ke visual studio code untuk segera di publish ke github dan di deploy oleh netlify, dan proses itu memang agak merepotkan sih. dan menurut saya lebih cocok digunakan untuk blog dibandingkan website besar.

## Kapan Anda merasa paling terinspirasi untuk menulis? 
saat muncul ide baru dan saat mood sudah membaik.

## Apakah Anda langsung mempublikasikan setelah menulis, atau membiarkannya sedikit "mengendap" sebagai draf? 
jika saya yakin maka saya akan langsung publish, tetapi semakin lama jika saya menambahkan gambar di dalam artikel saya, dan akan banyak revisi jika kata yang saya tulis agak riskan

## Apa postingan favorit Anda di blog Anda?
menurut saya postingan favorit saya tentang: masa lalu saya ketika menjadi santri karena poinnya banyak dan membuat saya bernostalgia dengan poin poin yang ada.

## Apakah ada rencana masa depan untuk blog Anda? 
Saya harus memperbaiki tulisan saya menjadi lebih enak untuk dibaca dan diresapi kata demi kata, dan yang pasti kualitas tulisanya diperbagus lagi.
tidak berhenti sampai disitu kuantitas tulisan pun ditingkatkan, entah itu jumlah kata per artikel atau jumlah postingan yang terupload setiap minggu. semoga bisa menjadi penulis hebat. 

## Mungkin mendesain ulang, pindah ke platform lain, atau menambahkan fitur baru?
