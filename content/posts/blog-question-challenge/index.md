## Sejak kapan anda mulai ngeblog?
Saya kurang begitu ingat tapi saya mulai membeli domain pertama saya pada tahun 2018 akhir dengan domain jundimubarok.com tapi karena semakin mahal, akhirnya saya pindah ke jundi.web.id yang 4x lebih murah dibanding ekstensi .com - Saya lupa tulisan pertama saya karena tidak ada arsip, ditambah lagi saya berpindah platform.

## Platform apa yang Anda gunakan untuk mengelola blog Anda dan mengapa Anda memilihnya?
Awal punya domain saya menggunakan platform wordpress yang memiliki tema yang berlimpah, plugin, serta kustomisasi yang fleksibel, bahkan saya sampai membeli 2 tema premium yang bisa diinstall di unlimited website. Sampai pada akhirnya saya tertarik untuk menggunakan hugo, platform generator website statis yang menggunakan bahasa Go dan itu gratis, walaupun pada akhirnya saya melihat dokumentasi karena pemahaman saya yang minim, sampai saat ini saya masih menggunakannya dengan alasan tidak ada database, sehingga tidak membebani server. Sebenarnya alasan yang utama adalah minimnya modal sehingga website statis menjadi pilihan dibanding dinamis.

## Apakah Anda pernah ngeblog di platform lain sebelumnya? 
Kalau platform untuk blogging hanya pernah di wordpress dan hugo, pernah mencoba menggunakan jekyll tapi tidak sebagus hugo untuk tema dan dokumentasinya.

## Bagaimana cara Anda menulis postingan? Misalnya, di alat pengeditan lokal, atau di panel/dasbor yang merupakan bagian dari blog Anda? 
Saya menuliskannya di dalam editor markdown, saya coba beberapa aplikasi yang cocok, pada akhirnya langsung dituliskan di dalam visual studio code, dan saat ini menggunakan google docs sehingga bisa dilanjutkan di smartphone jika tidak sedang menggunakan komputer, setelah itu barulah tulisan saya pindahkan ke visual studio code untuk kemudian di publish ke github dan di deploy oleh netlify, dan proses itu memang agak merepotkan sih, menurut saya lebih cocok digunakan untuk blog dibandingkan website besar yang mungkin menampilkan database yang besar.

## Kapan Anda merasa paling terinspirasi untuk menulis? 
Sebenarnya tergantung mood, walaupun itu sangat keliru, padahal tulisan saya lebih ke arah pengalaman bisa saja dibuat blog ini seperti journaling daripada sekedar curhatan belaka.

## Apakah Anda langsung mempublikasikan setelah menulis, atau membiarkannya sedikit "mengendap" sebagai draf? 
Jika saya yakin maka saya akan langsung publish, tetapi bisa semakin lama jika saya menambahkan gambar di dalam artikel saya, dan akan mengendap sebagai draft lebih lama jika tulisan saya cukup *depth*

## Apa postingan favorit Anda di blog Anda?
Menurut saya postingan favorit saya tentang: masa lalu saya ketika menjadi santri karena poinnya banyak dan membuat saya bernostalgia dengan poin poin yang ada.

## Apakah ada rencana masa depan untuk blog Anda? 
Saya harus memperbaiki tulisan saya menjadi lebih enak untuk dibaca dan diresapi kata demi kata, dan yang pasti kualitas tulisanya diperbagus lagi. Tidak berhenti sampai disitu kuantitas tulisan pun ditingkatkan, entah itu jumlah kata per artikel atau jumlah postingan yang terupload setiap minggu. Semoga bisa konsisten.

## Mungkin mendesain ulang, pindah ke platform lain, atau menambahkan fitur baru?
Mungkin iya, jika saya mau mempelajari css dkk. yang pasti saya tetap setia menggunakan hugo.
